## How to Use This Log

Every tester records feedback after each session. Use one row per issue, question, or observation. Be specific — "the share link page was confusing" is less useful than "the share link URL was not copied to clipboard automatically after clicking copy button."

---

## Feedback Template (per entry)

| Field | Description |
|-------|-------------|
| **Date** | YYYY-MM-DD |
| **Tester** | Name or alias |
| **Flow** | What were you testing? |
| **Type** | Bug / UX Issue / Confusion / Feature Request / Compliment |
| **Severity** | Blocker / High / Medium / Low |
| **Subject** | Short title |
| **Details** | Full description of what you found |
| **Action** | Fix / Deferred / Won't Do / Needs Discussion |
| **Owner** | Who will address it |
| **Status** | Open / In Progress / Done |
